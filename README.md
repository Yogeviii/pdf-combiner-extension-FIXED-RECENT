# Local PDF Combiner — Fixed Package

This package already includes `vendor/pdf-lib.min.js`.

## Load unpacked in Chrome

1. Unzip this package.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the unzipped `pdf-combiner-extension-FIXED` folder.
6. Click the extension icon.

Supported: JPG, JPEG, PNG, WebP, GIF first frame, BMP, AVIF when Chrome can decode it, and PDF.
