Commbox Conversation PDF Downloader

How to install:
1. Extract this ZIP.
2. Open Chrome: chrome://extensions
3. Enable Developer mode.
4. Click Load unpacked.
5. Select the extracted commbox_pdf_picker_extension folder.

How to use:
1. Open the relevant Commbox conversation.
2. Click the extension icon.
3. Click one PDF/message inside the target conversation.
4. The extension downloads all PDFs found inside that same conversation container.

Optional:
- Hold Alt while clicking the PDF/message to open the PDFs in background tabs instead of downloading them.

Notes:
- This extension is scoped to https://davidshield.commbox.io/* only.
- Files are saved under: Commbox PDFs/<conversation-id>_<date>/
